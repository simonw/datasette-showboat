Setup

To send showboat output to this server, set the SHOWBOAT_REMOTE_URL environment variable to point to the receive endpoint:

export SHOWBOAT_REMOTE_URL="https://your-datasette-instance{{ base_url }}-/showboat/receive"

If a token is configured, include it as a query parameter:

export SHOWBOAT_REMOTE_URL="https://your-datasette-instance{{ base_url }}-/showboat/receive?token=your-token"
